# Dataset Instructions

This case study uses the Fake and Real News Dataset from Kaggle.

Dataset link: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

To reproduce the project, download the dataset from Kaggle and place the dataset files in this `Data/` folder before running the scripts.

The expected files are usually:

- `Fake.csv`
- `True.csv`

These files contain fake and real news articles used for the text classification workflow.
